**Setup Instructions**  
1. Clone repository:
```bash
git clone https://github.com/MykeHaunt/NeuraLinkTech-AI-Vehicle-Diagnostics-nextgen
cd NeuraLinkTech-AI-Vehicle-Diagnostics-nextgen
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Configure hardware:
```bash
./scripts/setup_can.sh --profile a340e --bitrate 500000
```

4. Start system:
```bash
python -m neuralink start --mode production
```
